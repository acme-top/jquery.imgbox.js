## Introduction 介绍

jquery.imgbox.js是一款简单、轻量的imgbox插件，主要参考了简述的imgbox功能，支持图片缩放、拖曳功能。

## Docs 文档

```javascript

  $("img").imgbox({
	zoom: true,
	drag: true
  });

```
